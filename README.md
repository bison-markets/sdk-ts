# Bison SDK (TypeScript)

TypeScript SDK for Bison Markets.

## Installation

```bash
npm install bison-sdk-ts
```

**Requirements:**

- Node.js >= 22 (for native WebSocket support)
- All modern browsers

## Development

```bash
# Install dependencies
npm install

# Build
npm run build

# Development mode (watch)
npm run dev

# Generate API types from OpenAPI spec
OPENAPI_URL=... npm run generate:openapi

# Run tests
npm test

# Run tests in watch mode
npm run test:watch

# Lint and format
npm run lint
npm run format

# Type check
npm run typecheck
```
